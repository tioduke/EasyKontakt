EasyKontakt Export Import
EasyKontakt kann Telefon/Adressbücher aus anderen Programmen importieren, die nicht dem vCard Standard entsprechen.

Zwischenablage

Das VCard Format ist als standardisierte Schnittstelle zum Datenaustauch konzipiert. Deshalb eignet es sich auch dazu Kontaktdaten zwischen Programmen per Kopieren und Einfügen über die Zwischenablage des Betriebssystems auszutauschen.

Fritz.Box

Das Telefonbuch einer Fritz.Box kann in einem XML Dateiformat gesichert werden. Für dieses Fritz.Box XML Format ( getestet mit Fritz.Box 7170 und Fritz.Box 7270 ) enthält EasyKontakt ein Importmodul.

Thunderbird E-Mail

Das Thunderbird Adressbuch kann auf verschiedene Arten exportiert werden. In EasyKotakt ist ein Importmodul für eine Komma getrennte CSV Datei.
Diese CSV Datei enthält in des ersten Zeile die "Überschriften" zu den einzelnen Datenfeldern. Eine Zuordnung der Überschriften zu den im VCard Standard definierten Datenfeldern erfolgt in der Konfigurationsdatei thunderbird.def aus der hier ein kleiner Auszug folgt. 
Vorname=N:1 
Nachname=N:0:END
Anzeigename=FN
Spitzname=?
Primäre E-Mail-Adresse=EMAIL:PREF
Sekundäre E-Mail-Adresse=EMAIL
Messenger-Name=?
Tel. dienstlich=TEL:WORK
Tel. privat=TEL:HOME
Fax-Nummer=FAX
Als Beispiel nehmen wir die Zeile Nachname=N:0:END
Nachname
Das ist die Überschrift aus der CSV Datei von Thunderbird
N
Das VCard Datenfeld für den Namen
0
VCard definiert für den Namen 5 Einzelfelder, Die Position "0" entspricht dem Nachnamen
END
Dies markiert das letzte Einzelfeld aus den Datenfeld N wie Name

Durch diese flexible Zuordnung kann das Importmodul von EasyKontakt leicht auf Änderungen in der CSV Datei angepasst werden, wie z.B. dann wenn Thunderbird in einer anderen Landessprache benutzt wird.
